DNA Helix

Affichage lumineux d'un génome stocké dans un fichier

Nécessite : 
- arduino nano
- 2x ruban LED WS
- Lecteur de carte SD/microSD compatible avec la bibliothèque SD.h
- potentiomètre 100k

Le fichier doit s'appeler "genome.txt".
